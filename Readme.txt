本課題は、コーディング練習課題です。

「images」フォルダの画像を使用して、「pc_top.jpg」と同様のレイアウトになるように
WEBページのコーディングを行ってください。

※コーダーの業務にはpsdファイルからの画像切り出し（スライス）が含まれますが、
　今回はコーダーのPCにPhotoshopがインストールされていないケースを想定して
　スライスした画像を用意しています。



■制作範囲
PC版トップページ（1P）

■対象ブラウザ
IE9、IE11、Edge　※IEのブラウザモードを使用
Firefox最新版
Chrome最新版

■対象外環境
ゲーム端末
Amazon端末
テレビ
モバイル端末（フィーチャーフォン）
アプリ内ブラウザ
Opera、LunascapeなどのWEBブラウザ
各ブラウザからの印刷

■SEO
・最適と思われるmeta情報、titleを記述する
・最適な論理構造でコーディング記述を行う

■実装環境
HTML5、CSS3

■提出方法

データは【自分のGoogleドライブに保存し】、
URLならびに以下をメールに記載して、教育担当まで提出してください。

件名：【課題提出】コーディング課題（セラク銀行）　氏名

１：GoogleドライブURL（またはオンラインストレージURL）
２：使用ツール
３：制作時間（例：12h）
４：特に時間がかかった部分
５：この課題で学んだこと
